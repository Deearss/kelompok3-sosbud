Aturan mainnya apa?

Format, ketentuan referensi, tools, dan checklist final sebelum dikumpulkan.

Format Dasar
Elemen Ketentuan
Kertas HVS Kuarto (A4)
Font Times New Roman
Ukuran Font 12
Spasi 2 (double spacing) di seluruh dokumen
Daftar Pustaka APA Style
Struktur Dokumen
1. Cover → 2. Kata Pengantar → 3. Daftar Isi → 4. BAB I → 5. BAB II → 6. BAB III → 7. Daftar Pustaka
Ketentuan Referensi
Sumber Status Keterangan
Jurnal Akademik ✅ Utama Diutamakan — bobot tertinggi
Buku Teks / Ilmiah ✅ Utama Bisa dijadikan referensi utama
Blog / Wikipedia ⚠️ Bahan awal Boleh untuk riset awal, tapi jangan jadi sumber utama
Ketentuan Konten
  • Setiap klaim/pernyataan wajib disertai referensi akademik
  • Sertakan minimal 1 contoh kasus nyata yang relevan dan bisa dipertanggungjawabkan
  • Analisis tidak boleh dangkal — harus ada penjelasan "mengapa" dan "dampaknya"
  • Struktur argumen harus logis dan tidak ada celah antar bagian
Tools Kolaborasi
Kebutuhan Tools

Presentasi Google Slides
Menyimpan semua file Google Drive
